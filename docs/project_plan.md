# CiteMind Project Plan

## Goal

Build an AI-powered research paper assistant that can answer questions from uploaded PDFs with citations.

## Main Pipeline

1. Upload PDF
2. Extract text
3. Split text into chunks
4. Convert chunks into embeddings
5. Store embeddings in a vector database
6. Retrieve relevant chunks based on a user question
7. Generate an answer using an LLM
8. Return the answer with citations
