Endpoints

Available API routes

GET /health

Health Check

Returns the backend service status.

{
  "status": "healthy",
  "service": "CiteMind API"
}
POST /upload-pdf

Upload and Index PDF

Uploads a PDF, extracts text, chunks content, embeds chunks, and stores them in ChromaDB.

Form Data:
file: research_paper.pdf
POST /search

Similarity Search

Retrieves the most relevant chunks for a user question.

{
  "question": "What is the main idea?",
  "top_k": 3
}
POST /ask

Ask with Citations

Generates a grounded answer and returns clean citation labels.

{
  "question": "What problem does this paper solve?",
  "top_k": 3
}
GET /vector-store/stats

Vector Store Stats

Shows the current ChromaDB collection and indexed chunk count.

{
  "collection_name": "citemind_papers",
  "total_chunks": 42
}
