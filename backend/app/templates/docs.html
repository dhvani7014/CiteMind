POST /upload-pdf

Upload Papers

Accepts a PDF file, extracts text, chunks the content, creates embeddings, and stores the paper in the vector database.

POST /ask

Ask Questions

Takes a user question, retrieves relevant paper sections, and returns an answer with citations and source metadata.

POST /search

Search Sources

Returns the most relevant chunks from the active document without generating a full AI answer.

GET /vector-store/stats

View Stats

Shows the active document and how many chunks are currently indexed in the vector store.
